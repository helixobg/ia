# Project Bac Pro Ia générative pour le text

### Présentation

Astra Ia est une ia où on peux discuter avec elle.
Elle a était créer pour un project pour une épreuve de mon bac pro
Elle permet d'aider à faire des tp, elle cherche des documents sur le serveur du lycée et bien plus.

### Temporaire

Rien n'est encore mis cela est normal :)

### Liste des Fichiers

- app.py
- tokenizer.py
- model.py
- train.py
- requirements.txt

### Comment tester l'ia

En tout premier télécharger les fichiers.

Ensuite il faut installer les dépendances pour que cela fonctionne :

```pip install requirements.txt```

Voici ce qu'il faut faire dans l'ordre pour que cela fonctionne après avoir installer les dépendances :

```
python tokenizer.py
python model.py
python train.py
python app.py
```
